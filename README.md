#FitList

##Goal  
A compendium of complete workout rountines that can be voted on, rated, and discussed by the community.

##Development Tools 
* Python3
* Flask
* SQLite
* React.js

##Core Features 
* Workout rountes complete with spreadsheets
* User reviews
* Comments
* Upvote/Downvote system
* Videos/photos
* Tags/Categories
* Community profiles
